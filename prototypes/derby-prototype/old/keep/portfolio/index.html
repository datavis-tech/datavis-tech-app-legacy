Portfolio

Datavis Tech Inc. was founded by Curran Kelleher in January 2016. This selection of Curran's past projects demonstrates the flavor of the technology, products and services that Datavis Tech will offer.

UserInsight Ingress Dashboard

Interactive visualization dashboard with multiple linked views for exploring and presenting login data for enterprise computer networks. This work is in production in the Rapid7 product UserInsight, cybersecurity data analysis product. Here's a video of this work in action in the product.

World Population Visualization

This interactive data visualization lets you explore the population of countries over time between 1950 and 2010. There are two linked views, and interaction in each controls the filter applied to the data shown in the other. Hovering in the line chart filters the map data by year. Zooming in the map causes the line chart to show only the visible countries.

Open Source Visualizations

A collection of data visualization examples hosted on bl.ocks.org, the canonical host for examples demonstrating use of D3.js. These numerous examples are ready to be used by anyone as starting points for their visualizations.
