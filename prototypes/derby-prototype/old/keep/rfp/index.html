Datavis Tech Inc. offers consulting services for design and development of interactive data visualizations. We specialize in small, short-term projects in which your data is visualized for the Web using D3 and other Open Source modules. Given a data set or API to work with, we can help you explore and present the data visually. During the beginning of a project, we can try out a number of visual encodings and interaction techniques for your data. Towards the end of a project, we can work together to evolve the deliverable depending on your target audience and the types of tasks you wish to enable using data visualization.

For us to quickly evaluate possible approaches to visualizations, it's helpful to include a sample of data in your submission. This should be in Comma Separated Value (csv) format, and should have a tabular structure that resembles the data that you'd like to visualize. This can be synthetic, anonymized, or public data.
